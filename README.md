# Cape Town Food Fest Ticket Landing Page

## Project Overview

This project is a Vue 3 single-page application that showcases ticket tiers for the Cape Town Food Fest. The landing page displays Bronze, Silver, and Gold ticket options with pricing, descriptions, benefits, and a favourite feature to simulate user engagement.

## Features

* Dynamic rendering of ticket tiers using Vue components
* Reusable Ticket Card component
* Featured ticket highlighting
* Favourite button interaction
* Responsive design for desktop and mobile devices
* Modern event landing page layout

## Technologies Used

* Vue 3
* Vite
* JavaScript
* CSS

## Installation and Setup

### Clone the Repository

```bash
git clone https://github.com/YOUR-USERNAME/food-fest-ticket-landing-page.git
```

### Navigate to the Project Folder

```bash
cd food-fest-ticket-landing-page
```

### Install Dependencies

```bash
npm install
```

### Run the Development Server

```bash
npm run dev
```

### Open in Browser

After running the command above, Vite will display a local URL similar to:

```text
http://localhost:5173/
```

Open the URL in your browser to view the application.
